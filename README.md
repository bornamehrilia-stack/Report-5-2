# Report-5-2

## ۱. مقدمه  
در این پروژه، یک **نمایشگر هفت قطعه‌ای** (Common Cathode) به آردوینو متصل شده است و با استفاده از کد، **حروف A, C, E, F, H, L, P** به صورت دیجیتال و به ترتیب نمایش داده می‌شوند..

## ۲. اهداف پروژه  
- آشنایی با ساختار نمایشگر هفت قطعه‌ای و پین‌های آن  
- یادگیری نحوه‌ی کنترل هر قطعه با استفاده از پین‌های دیجیتال  
- استفاده از آرایه‌ها برای ذخیره الگوهای حروف  
- نمایش حروف A, C, E, F, H, L, P به صورت متوالی  
- آشنایی با مفهوم “کد باینری” و “نمایش دیجیتال”

## ۳. قطعات مورد استفاده  
- برد آردوینو Uno  
- نمایشگر هفت قطعه‌ای (Common Cathode)  
- ۷ عدد مقاومت ۲۲۰ اهم (برای هر قطعه یک عدد)  
- برد برد (Breadboard)  
- سیم‌های جامپر (jumper wires)



## ۴. نحوه‌ی ساخت مدار  
- **نمایشگر هفت قطعه‌ای**:  
  - هر قطعه (a, b, c, d, e, f, g) → به یکی از پین‌های دیجیتال ۱ تا ۷ آردوینو متصل شده است.  
  - هر قطعه با یک مقاومت ۲۲۰ اهم به GND متصل شده است.  
  - پین مشترک (Common Cathode) → به GND متصل شده است.



## ۵. کد برنامه:

```cpp
// الگوهای هفت قطعه‌ای برای حروف A, C, E, F, H, L, P
int sequence[][7] = {
  {1, 1, 1, 0, 1, 1, 1}, // A
  {1, 0, 0, 1, 1, 1, 0}, // C
  {1, 0, 0, 1, 1, 1, 1}, // E
  {1, 0, 0, 0, 1, 1, 1}, // F
  {0, 1, 1, 0, 1, 1, 1}, // H
  {0, 0, 0, 1, 1, 1, 0}, // L
  {1, 1, 0, 0, 1, 1, 1}  // P
};

// تابع نمایش یک حرف بر اساس شماره
void printNumber(int number) {
  for (int i = 0; i < 7; i++) {
    digitalWrite(1 + i, sequence[number][i]);
  }
}

void setup() {
  // تنظیم پین‌های ۱ تا ۷ به عنوان خروجی
  pinMode(7, OUTPUT);
  pinMode(6, OUTPUT);
  pinMode(5, OUTPUT);
  pinMode(4, OUTPUT);
  pinMode(3, OUTPUT);
  pinMode(2, OUTPUT);
  pinMode(1, OUTPUT);
}

void loop() {
  // نمایش هر حرف به ترتیب
  for (int i = 0; i < 7; i++) {  // ۷ حرف
    printNumber(i);
    delay(1000);  // نگه‌داشتن هر حرف برای ۱ ثانیه
  }
}
```

### توضیح کد:  
- `sequence[][7]` — یک آرایه دوبعدی است که الگوهای هر حرف را ذخیره می‌کند.  
- `printNumber(int number)` — یک تابع است که بر اساس عدد ورودی (شماره حرف)، الگوی مربوطه را روی نمایشگر نمایش می‌دهد.  
- `digitalWrite(1 + i, sequence[number][i])` — هر قطعه را بر اساس الگو روشن یا خاموش می‌کند.  
- `for (int i = 0; i < 7; i++)` — حروف A تا P را به ترتیب نمایش می‌دهد.  
- `delay(1000)` — برای اینکه هر حرف برای ۱ ثانیه نمایش داده شود.

## ۶. نحوه‌ی کارکرد  
- پس از آپلود کد، نمایشگر شروع به نمایش حروف A, C, E, F, H, L, P می‌کند.  
- هر حرف برای ۱ ثانیه نمایش داده می‌شود و سپس به حرف بعدی تغییر می‌کند.  
- این الگو به صورت بی‌پایان تکرار می‌شود.

## ۷. نتایج  
نمایشگر به درستی حروف A, C, E, F, H, L, P را به ترتیب نمایش می‌دهد..
